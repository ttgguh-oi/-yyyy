# 智能瞭望系统演示脚本

## 演示时长
1-3分钟

## 演示目标
展示智能瞭望系统的核心功能：用户注册登录、多源数据抓取、数据存储、数据清洗和分析。

## 演示环境准备
1. 确保系统已正确安装并启动
2. 打开浏览器，访问 http://127.0.0.1:5000

## 演示步骤

### 1. 用户注册与登录 (30秒)

1. **注册新用户**
   - 打开浏览器访问系统首页
   - 点击右上角"注册"按钮
   - 填写注册信息：
     - 用户名：demo_user
     - 密码：demo_password
     - 确认密码：demo_password
   - 点击"注册"按钮完成注册

2. **登录系统**
   - 返回首页点击"登录"按钮
   - 输入用户名和密码登录
   - 登录成功后进入仪表盘

### 2. 数据抓取功能演示 (45秒)

1. **添加网页数据源**
   - 点击左侧菜单"数据抓取"
   - 点击"添加数据源"按钮
   - 填写数据源信息：
     - 名称：示例新闻网站
     - 类型：网页(Web)
     - URL：https://www.example.com
     - 配置：
       ```json
       {
         "selector": ".news-item",
         "title_selector": "h3",
         "content_selector": ".news-content"
       }
       ```
   - 点击"保存"按钮

2. **添加API数据源**
   - 点击"添加数据源"按钮
   - 填写数据源信息：
     - 名称：示例API
     - 类型：API接口
     - URL：https://jsonplaceholder.typicode.com/posts
     - 配置：
       ```json
       {
         "method": "GET",
         "items_key": "",
         "title_path": "title",
         "content_path": "body"
       }
       ```
   - 点击"保存"按钮

3. **执行数据抓取**
   - 在数据源列表中，点击API数据源的"爬取"按钮
   - 等待爬取完成
   - 点击"查看数据"按钮，展示抓取结果

### 3. 数据处理与分析功能演示 (45秒)

1. **数据清洗**
   - 点击左侧菜单"数据分析"
   - 在"数据清洗"区域：
     - 选择数据源：示例API
     - 勾选清洗选项：移除重复数据、移除空内容、文本标准化
     - 点击"开始清洗"按钮

2. **数据分析**
   - 在"数据分析"区域：
     - 选择数据源：示例API
     - 分析类型：文本分析
     - 点击"开始分析"按钮

3. **查看分析结果**
   - 等待分析完成
   - 在"最近分析结果"中点击"查看结果"按钮
   - 展示分析结果，包括词频统计、关键词等

### 4. 系统概览与总结 (30秒)

1. **仪表盘查看**
   - 点击左侧菜单"仪表盘"
   - 展示系统概览信息：数据源总数、爬取数据量、分析结果数
   - 展示任务状态和最近活动

2. **总结**
   - 强调系统的核心价值：多源数据统一管理、智能分析
   - 指出系统的可扩展性：支持更多数据源类型和分析方法

## 功能亮点说明

1. **多源数据支持**
   - 网页、API、本地文件三种数据源类型
   - 统一的数据管理界面

2. **智能数据处理**
   - 自动数据清洗功能
   - 多种数据分析方法：基础统计、文本分析、情感分析等

3. **友好的用户界面**
   - 基于Bootstrap的响应式设计
   - 直观的数据可视化展示

4. **可扩展架构**
   - 模块化设计，便于功能扩展
   - 支持自定义数据源和分析方法

## 演示注意事项

1. **准备测试数据**
   - 可以提前准备几个简单的测试数据源，确保演示流畅
   - 建议使用公开的、稳定的API接口进行演示

2. **控制演示时间**
   - 每个步骤严格控制时间，避免超时
   - 对于耗时较长的操作（如数据爬取、分析），可以提前准备好数据

3. **突出核心功能**
   - 重点演示系统的核心功能：数据抓取、处理和分析
   - 避免过多的细节操作，保持演示简洁明了

4. **展示界面美观**
   - 保持浏览器窗口整洁，关闭不必要的标签页
   - 确保系统界面显示完整，避免滚动条过多

## 备选演示方案

如果网络条件不佳或演示时间有限，可以采用以下备选方案：

1. **使用本地文件数据源**
   - 提前准备一个CSV或JSON格式的本地文件
   - 在演示中使用本地文件作为数据源，避免网络延迟

2. **预先生成分析结果**
   - 在演示前提前运行一次数据分析
   - 演示时直接查看预先生成的分析结果

3. **简化注册流程**
   - 如果时间紧张，可以跳过注册步骤，直接使用预设的测试账号登录

## 技术要点回顾

1. **多源数据抓取技术**
   - 使用Requests库处理HTTP请求
   - 使用BeautifulSoup4解析网页内容
   - 支持API接口和本地文件读取

2. **数据存储技术**
   - 使用SQLAlchemy ORM框架
   - 支持SQLite和MySQL数据库
   - 数据统一存储和管理

3. **数据分析技术**
   - 使用Pandas进行数据处理
   - 使用scikit-learn进行机器学习分析
   - 使用jieba进行中文分词

4. **用户界面技术**
   - 使用Flask框架搭建Web应用
   - 使用Bootstrap 5构建响应式界面
   - 使用Font Awesome图标增强视觉效果
